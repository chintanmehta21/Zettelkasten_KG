What Is It?

Your second brain, with memory that compounds.

Zettelkasten turns scattered links into connected notes so ideas stop living in tabs and start living in a system.

What It Is

Paste a URL from YouTube, GitHub, Reddit, newsletters, or any web page. The system extracts signal, writes a structured summary, tags it, and places it inside your connected knowledge graph.

Instead of isolated bookmarks, every capture becomes reusable context.

  3. 3

  4. 4
