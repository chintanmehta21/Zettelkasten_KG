About

Turn a link into a memory you can return to.

Zettelkasten captures YouTube videos, GitHub repos, Reddit threads, newsletters, and normal web pages. It extracts the signal, writes a structured summary, and adds the note to a connected graph so the next capture has context.

Instead of another bookmark folder, you get a living vault. The note is searchable, the source is preserved, and the relationships are discoverable later.

How it works

  1. 01
    Capture

    Drop in a URL from the web, YouTube, GitHub, Reddit, or a newsletter.

  2. 02
    Understand

    The source is detected and summarized with useful tags and a clean note structure.

  3. 03
    Connect

    Related notes are linked automatically so each new idea grows your graph.

  4. 04
    Revisit

    Search, sort, and open your captures later without rebuilding the context.

Features

01

One-Tap Capture

02

Your zettels stay searchable and sortable so the vault stays practical even as it grows.

06

Designed For Weak Connections

The web UI stays light, the browser cache stays tiny, and the experience keeps moving on slower networks.
